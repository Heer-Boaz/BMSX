# BMSX Lua Shell

This rompack boots directly into the BMSX Console editor with an empty Lua cart.

- Edit `res/lua/shell.lua` or write new Lua code from the in-engine editor.
- Press `ESC` to toggle the editor, `SAVE` to persist, and `REBOOT` or `RESUME` to execute the updated code.
- The default runtime draws a BASIC-inspired splash screen to confirm the shell is active.
